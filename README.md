# string.distance

Library to support implementations for the calculation of distances between strings according to several algorithms


#### <i class="icon-refresh"></i> **Algorithms implemented:**

>  Optimal Alignment
>  
> Damerau - Levenshtein
>
> Needleman - Wunsch
>
> Smith - Waterman

```flow
st=>start: Start
e=>end
op=>operation: Calculate distance

st->op->e


```
